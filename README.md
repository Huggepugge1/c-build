# c-builder
A simple C project builder for Linux.

## Features
 - [x] Initialize a new C project
 - [x] Build the project
 - [x] Run the project
 - [x] Run the project with memory checks
 - [x] Clean the project
 - [] Run tests on the project

## Usage
### Initializing a new project
```bash
c-builder init <project-name>
```
